# CauldronBrew
_Spigot plugin that can make things in cauldron_

# Config file
```yml
Localization: en

#Main Settings of cauldron
Cauldron:
  secondstoboil: 10
  blocktoupgrage: iron_block
  maxitemsincauldron: 32

#Event PlayerDropItem and checks where item is to drop in cauldron
dropitem:
  checknearbycauldron:
    delay: 5
    radius: 0.1
    secondstocheck: 20

#Holograms in SUPERALPHA!!!!
Hologram:
  enabled: false
  visiblinradius:
    enabled: true
    radius: 10
  floating:
    enabled: true
    distance: 0.2

#dont touch
configversion: 0.1
```
